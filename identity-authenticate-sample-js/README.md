# WSO2 Identity Server - OIDC SDK Vanilla Javascript Usage Example

Vanilla JS authentication usage sample app

---

## Getting started

1. Run `yarn run build`
2. run `yarn serve`
3. Visit `http://localhost:5000` on browser
